# Finite Limit
$$
\lim_{x\to a}f(x) = L \Leftrightarrow  \forall \varepsilon > 0,  \exists \delta > 0 \\ 
\quad \text{s.t} \quad \\
0 < |x-a| < \delta \implies 0 < |f(x)-L| < \epsilon 
$$
